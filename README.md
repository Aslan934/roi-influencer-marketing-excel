## 📌 Project Overview

This project transforms raw influencer marketing data into a structured, interactive Excel dashboard. The goal was to go beyond vanity metrics (likes, views) and answer a harder question: **which influencer investments are actually driving sales?**

The dashboard enables users to slice and compare performance across influencers, platforms, campaign types, and product categories — making it easy to spot what to scale and what to cut.

---

## 🔍 Key Insights

- **Sales are stable but not scaling** — the current strategy is functional, but lacks the momentum needed for growth. This signals a need to revisit budget allocation or campaign diversity.
- **High engagement, weak conversion** — traffic and interaction are strong, but they aren't translating into purchases. A gap exists between audience interest and buying intent.
- **Platform performance is uneven** — not all platforms deliver equal value. Some channels consistently outperform others, pointing to where future spend should be concentrated.
- **Campaign type matters** — certain formats (e.g. giveaways, product-focused campaigns) significantly outperform others, showing that content strategy directly impacts ROI.
- **Product category imbalance** — some categories drive the majority of sales while others consistently underperform, indicating a need for more targeted influencer-category matching.

## 📋 Dashboard Features

| Section | What It Shows |
|---|---|
| ROI by Influencer | Which influencers generate the most return relative to cost |
| ROI by Platform | Platform-level performance comparison |
| Engagement Rate Analysis | Reach and interaction metrics across campaigns |
| Campaign Type Breakdown | ROI and sales by campaign format |
| Sales Performance | Revenue trends and category-level contribution |
| KPI Summary Cards | Top-line metrics at a glance |

Interactive slicers allow filtering by platform, campaign type, influencer, and product category.
